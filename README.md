# HAI学伴系统 (Human with AI)

[!\[GitHub\](https://img.shields.io/badge/GitHub-Repository-blue?logo=github null)](https://github.com/xu-wen-yu/HAI)

## 🎯 项目简介

HAI学伴是一个智能学习伙伴系统，以聊天系统的形式构建学习者与AI学伴之间的互动。学伴可以解答学习者的专业问题，提供24小时不间断的学习支持。

**GitHub 仓库**: [https://github.com/xu-wen-yu/HAI](https://github.com/xu-wen-yu/HAI)

## 🌟 核心特色

### 🤖 AI智能对话

- **24小时在线学习伙伴**：随时解答专业问题
- **基于RAG技术**：结合知识库提供个性化服务
- **多模式回复**：根据用户类型提供不同深度的回答
- **对话历史**：保存和查看历史对话记录

### 🔒 完全匿名

- 学伴不能透露身份，学习者也不能透露身份
- 保护用户隐私，专注于学习内容本身
- 支持匿名快速体验功能

### 👥 用户管理系统

- **多角色支持**：用户、学习者、导师三种角色
- **用户类型**：免费用户和付费用户
- **学习者档案**：详细的学习偏好、目标、风格设置
- **功德积分系统**：免费用户通过互动获得积分

### 📚 智能知识库

- 内置多领域专业知识
- 支持RAG技术进行知识检索
- 向量相似度匹配
- 持续更新和扩展知识内容

### 📊 学习管理

- **学习目标设定**：设置和管理个人学习目标
- **进度跟踪**：实时更新学习进度和掌握程度
- **数据可视化**：学习数据的图表展示和分析
- **学习笔记**：记录和管理学习笔记

## 🚀 快速开始

### 环境要求

- Python 3.8+
- pip包管理器

### 选择界面版本

项目提供两个界面版本，您可以选择其中一个使用：

#### 版本 1: Flask 版本（推荐，原来的界面）

这是您熟悉的界面，使用 HTML 模板。

**启动命令：**

```bash
python app.py
```

或使用 Python 3.13（推荐）：

```bash
py -3.13 app.py
```

**访问地址：**

- [http://127.0.0.1:5000](http://127.0.0.1:5000)
- 或 [http://localhost:5000](http://localhost:5000)

### 安装依赖

首次运行前，请安装依赖包：

```bash
pip install -r requirements.txt
```

### 初始化知识库（可选）

如果需要初始化示例知识库：

```bash
python init_knowledge.py
```

### 配置 API（可选）

如需使用真实的 AI 回复，可以在学习者中心配置 API：

1. 进入"学习者中心"
2. 找到"API配置"部分
3. 选择 AI 提供商（OpenAI、DeepSeek、硅基流动等）
4. 输入 API Key 并测试连接
5. 选择可用模型即可开始使用真实 AI

**支持的 API 提供商：**

- **OpenAI**：GPT-3.5/4 系列模型
- **DeepSeek**：深度求索大模型
- **硅基流动**：国内可用的 OpenAI 兼容服务
- **Anthropic Claude**：Claude 系列模型
- **Ollama**：本地运行的大模型
- **自定义**：任何兼容 OpenAI 格式的 API 服务

## 📋 快速开始

### 用户注册

1. 选择用户类型：免费学伴或付费学伴
2. 填写用户名和密码
3. 开始使用AI学伴

### 匿名体验

- 点击"匿名体验"按钮快速体验
- 无需注册即可使用基础功能

### 聊天功能

1. 在输入框中输入学习问题
2. AI学伴会基于知识库提供回答
3. 支持多轮对话和上下文理解
4. **开启知识库检索**：在对话界面开启知识库检索开关，AI 会优先从上传的文档中检索答案
5. **选择模型**：在对话界面底部选择不同的 AI 模型进行对话

### 知识库管理

1. 进入"知识库管理"页面
2. 拖拽或选择文件上传（支持 PDF、Word、Markdown、TXT）
3. 开启知识库检索后，AI 会从上传的文档中检索答案
4. 查看已上传文档的列表和统计信息
5. 删除不需要的文档

### API 配置

1. 进入"学习者中心"
2. 找到"API配置"部分
3. 选择 AI 提供商（OpenAI、DeepSeek、硅基流动、Anthropic Claude、Ollama等）
4. 输入 API Key 并点击"测试连接"
5. 连接成功后选择可用模型
6. 在对话界面即可使用配置的 API 进行真实 AI 对话

## 🏗️ 技术架构

- **后端框架**：Flask
- **前端技术**：Bootstrap 5, jQuery, Font Awesome
- **数据库**：SQLite
- **机器学习**：scikit-learn (RAG向量检索)
- **AI模型**：OpenAI API (可选，可配置)
- **文档处理**：LangChain, PyPDF, python-docx, Markdown
- **向量数据库**：FAISS
- **嵌入模型**：sentence-transformers (多语言支持)

### 核心算法

- **RAG（检索增强生成）**：结合知识检索和内容生成
- **TF-IDF向量化**：文本特征提取
- **余弦相似度**：知识库内容匹配
- **语义嵌入**：使用 sentence-transformers 进行多语言语义理解
- **智能分块**：基于语义的文档分块策略

## 🛡️ 安全特性

- **密码加密**：使用Werkzeug进行安全密码哈希
- **会话管理**：安全的用户会话和状态管理
- **数据验证**：输入数据的完整性和有效性检查
- **错误处理**：完善的异常处理和错误提示
